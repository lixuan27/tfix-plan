Every clip below runs through one set of weights that has seen clean video and its own reconstruction, and that has seen zero edited video.

The recipe behind these results

Editor, frozen: Qwen-Image-Edit, four step distilled int4, run per frame.

Fixer, trained: Wan2.1 causal 1.3B, source withheld, distribution matching on top of reconstruction data. dmfix_sfree_recon_S80/fixer_000500.pt, four steps.

Training data: 1028 reconstruction pairs across dashcam, ego, people and nature. Each pair is a clean clip and its own per frame reconstruction under a vague prompt such as reconstruct the same image.

01   the task we actually train on

Reconstruction: the editor reproduces a clean clip, and the fixer settles it.

This is the training objective, evaluated on clips held out of training. The editor is asked to change nothing, and because it draws every frame on its own, it returns a copy that is structurally faithful and temporally unstable. The fixer carries that copy back to a coherent video.

02   the ability that arrives for free

Editing: the same weights, a real instruction, and zero editing supervision.

Nothing about the fixer changes here. The frozen editor simply receives a real edit prompt instead of a reconstruction prompt, and the fixer performs the operation it already learned: hold the appearance, repair the time.
